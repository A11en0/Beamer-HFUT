# Beamer-HFUT
合肥工业大学beamer幻灯片模板
在使用PPT的过程中，确实难以忍受排版的痛苦，这是为合肥工业大学情感计算研究所，自然语言处理团队制作的beamer的幻灯片模板，参考了beamer-fudan和sgp940的工作（基本上是拷贝和整合），可以显示中文和英文，请各位自由修改和使用。Xelatex直接编译example.tex运行，在Mac系统下MacTex2018调试通过。
